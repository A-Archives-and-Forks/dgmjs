# dgmjs

An extensible diagramming framework for the web

## Build and publish

```sh
cd packages/dgmjs
npm run build
npm publish
```

## Run demo

```sh
npm run dev -w apps/demo
```

## Dev mode

```sh
npm run watch -w @dgmjs/core
npm run dev -w demo
```
