# dgm.js

An infinite canvas with _smart shapes_. It supports:

- 💡 Smart shapes (scripting, constraints, extended properties)
- 🔧 Headless components
- 🔥 Infinite canvas
- 📑 Multi-page support
- 👍 Hand-drawn styles
- 👥 Real-time collaboration
- 🎨 Dark mode (adaptive colors)
- 📸 Export to image (PNG, SVG)
- 🔤 Rich text
- 🧑🏻‍💻 JSON export/import
